Thinking in Angular 2

What do we need to know to google how to make an Angular 2 project

Terminology:

Components: Angular 2 breaks down features of the app and or pieces of your view into these blocks called components

Modules: Modules are groupings of the Components

Terminology continued:

Services: In order to access outside data inside all of your little components Angular 2 uses services. This is a way to have persitant data around the app

Data-binding: This is similar to Angular 1's data-binding, but differnet in syntax and usage. Data-binding in Angular 2 ins is used heavily to ______ * maybe it's not that different

For a full over view of the Angular 2 architecture check this out: https://angular.io/docs/ts/latest/guide/architecture.html

Okay so lets start making changes to the blank app. All the changes we make will automatically update in the browser.


The first place to look in /src then /app directory at the app.component.ts file. This is the file we will be working with primarily today.

The AppComponent is the root component by default for the Angular 2 project. Since our project it small we done need to bother making another component and adding it to the root app module, which we'll look out in a bit.

At the top of every component that you write you must import component package from the angular core library


							import { Component } from '@angular/core';

The compents html selector(like directive in angular 1), link to styling, and templating are declared here in this @Component meta tag


							@Component({
  selector: 'my-app',
  template: `
Hello {{name}}
`,
})

This is more tags that can be added to our Meta tag that I have yet to explore but know that these are not the only ones

Outside of our Component tag we have a Typescript class that we are exporting so that this piece of view and logic can be acessed a through the app module and displayed on screen. In this class is where logic for the component is stored and variables for databinding and accessing your model are declared


	 						export class AppComponent  {
								name = 'Angular';
							}

first thing that we wanna add for our TODO app is a Todo class at the top of our app

export class Todo {
  isDone: boolean;
  itemName: string;
}

Next inside of the AppComponent Class we want to create Todo arrary of tasks that we can display when the app launches

The syntax for creating and intializing an array in Typescript is:


							todoList: Todo[] = [
						{isDone: false, itemName: "item 1"},
						{isDone: false, itemName: "item 2"},
						{isDone: false, itemName: "item n"}
						];
